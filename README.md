**Diabetic Retinopathy Detection Using Deep Learning**
**Project Overview**

This project implements a Convolutional Neural Network (CNN) using Transfer Learning (ResNet50) to classify retinal fundus images into five categories of Diabetic Retinopathy severity.

**Objective**

To develop an AI-based automated screening system for early detection of diabetic retinopathy.

**Model Architecture**

Transfer Learning: ResNet50

Image Size: 224x224

Optimizer: Adam

Loss Function: Categorical Crossentropy

Epochs: 5
